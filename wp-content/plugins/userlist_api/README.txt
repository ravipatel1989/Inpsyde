Here is a short description of the plugin.  
This is a user list plugin which displays user list from the thirdparty api

== Description ==

1. When we enable this plugin it will create a custom link called SITE_URL/my-lovely-users-table
2. We can navigate to this link through top button link
3. When we go to the link we can see the list of the users from https://jsonplaceholder.typicode.com/users api
4. When we click on user we can get user from https://jsonplaceholder.typicode.com/user/n API
5. Here n is the user id, and we can see the details of the user
